### 🛠️ Como Instalar
1. Faça o download do arquivo `.exe` acima.
2. Execute o instalador (caso o Windows SmartScreen apareça, clique em *"Mais informações"* e *"Executar assim mesmo"*).
3. O bot será instalado e aberto automaticamente.
4. **Auto-Update**: Quando houver uma nova versão, o bot notificará e baixará os arquivos automaticamente na próxima inicialização.
